## Tech Stack
- Next.js
- TypeScript
- Clerk.com


## Packages
```bash
> npm install @clerk/nextjs
> npx shadcn-ui@latest init
> npx shadcn-ui@latest add
> npm install next-themes
> npm i prisma --save-dev
> npm i @prisma/client
> npx prisma init
# set up schema.prisma
> npx prisma migrate
> npx prisma studio
> npx prisma db push
> npm i -D react-icons 
> npm install @dnd-kit/core
```

## To Run
```bash
npm run dev
```
